# 🏥 JCI Observation Tagger

**Automatically tag JCI survey observations to the correct Standard and Measurable Element (ME)**  
Based on **JCI Hospital Accreditation Standards, 8th Edition** (Effective 1 January 2025)

---

## What it does

Surveyors type their observations in plain language — the tool uses offline AI semantic search to instantly identify the **1–5 most relevant JCI Measurable Elements**, ranked by confidence.

- ✅ **270 MEs** across all 18 chapters (ACC, AOP, COP, ASC, MMU, PCC, QPS, GLD, FMS, SQE, MOI, PCI, IPSG, APR, GHI, and more)
- ✅ **Fully offline** — no API key, no internet required after setup
- ✅ **Batch mode** — process dozens of observations at once and export to CSV
- ✅ **Chapter filter** — narrow search to specific chapters (e.g. FMS only)
- ✅ **Free deployment** on [Streamlit Cloud](https://streamlit.io/cloud)

## Quick start

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/jci-obs-tagger.git
cd jci-obs-tagger
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Build the search index (one time only)
```bash
python setup.py
```
This downloads the ~80 MB AI model and creates `mes_index.pkl`. Takes ~60 seconds. Never needed again.

### 4. Launch the app
```bash
streamlit run app.py
```

## Deploying to Streamlit Cloud (free)

1. Push this repo to GitHub (make sure `mes_database.json` is committed)
2. Go to [share.streamlit.io](https://share.streamlit.io) → **New app**
3. Select your repo and set **Main file path** to `app.py`
4. Click **Deploy**

> **Note:** On Streamlit Cloud the model downloads and the index builds automatically on first launch (~60 sec). After that it's cached.  
> The `mes_index.pkl` file is **not** needed in the repo — Streamlit Cloud builds it fresh.

## How it works

| Step | What happens |
|---|---|
| **Setup** | All 270 JCI MEs are embedded using `sentence-transformers/all-MiniLM-L6-v2` (a free, offline semantic embedding model) |
| **Search** | Your observation is embedded with the same model and compared to all MEs using cosine similarity |
| **Ranking** | Top matches are returned with a confidence score (0–100%) |

The model understands **meaning, not keywords** — so "crash cart not inspected" correctly matches "The hospital maintains and tests medical equipment" even without shared words.
